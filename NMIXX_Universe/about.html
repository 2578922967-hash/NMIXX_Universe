About This Project

A Fan Letter in Code.

Project Overview

This website was created as a final assignment for the AI and Digital Media Workshop course. The goal was to design a responsive, visually appealing static website using only HTML and CSS.


Developer Note

I chose NMIXX as the subject because of their unique "MIXX POP" concept which provided a lot of inspiration for the visual design—blending contrasting elements like the deep navy background with bright pastel accents.
